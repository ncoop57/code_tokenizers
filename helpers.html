download_grammar

 download_grammar (language)

Download the tree-sitter grammars for the specified languages.

If the languages argument is the string ‘all’, all available grammars will be downloaded. Otherwise, the argument should be a list of language codes to download.

# test downloading the python grammar
lang_file = download_grammar("python")
assert lang_file.exists()
assert lang_file.name == "languages.so"
